面向对象

发表于2022-04-24更新于2022-05-03

类

例子

使用关键字 new 实例化对象。

类不占内存，对象 (类中的字段) 占内存。

类的结构的区别：结构面向过程，不能封装、不能继承、没有多态。

属性

没有办法限制对字段非法值的赋值。

属性可以保护字段，限定字段的取值和赋值。正常情况下，应当对每个字段赋予一个属性。

属性的本质就是两个方法，get{} 和 set{}. 所有和外界打交道的事全部交给属性，这也是赋值给属性的原因。

属性有 get 方法时具有可读属性，有 set 方法时具有可写属性。

Fields 字段；Methods 方法；Properties 属性

因此通过属性赋值可以限制赋值从而保护对象。注意，属性并没有存储值，属性的值都存储在字段中，详见源代码。

访问修饰符

  • public: 公开，在哪都可以访问。
  • private: 私有，只能在当前类的内部访问。

静态与非静态

非静态类

非静态类中既可以有静态成员，也可以有非静态成员 (实例成员)。

调用实例成员时，需要使用 对象名.实例成员名 的语法。
调用静态成员时，需要使用 类名.实例成员名 的语法。参考 Cosnole.WriteLine();

静态类

Cosnole 类是静态类。

总结

  • 静态成员必须用类名调用，实例成员必须用对象名调用。
  • 静态方法只能访问静态成员，不能访问实例成员。
  • 实例方法既可以访问静态成员，也可以访问实例成员。
  • 静态类中只能声明静态成员，无法声明实例成员。
  • 静态类无法实例化。

使用

  • 如果想要一个创建一个工具类，可以考虑创建为静态类 (例 Console 类)。这样就可以直接调用内部的静态成员，而不需要先实例化。
  • 静态类在整个项目中资源共享，只要程序在运行，静态类就不会释放资源，因此项目中的静态类不宜过多。

堆、栈、静态存储区
静态成员都存储在静态存储区，在整个项目中，谁都可以访问静态存储区，因此静态类在整个项目中资源共享。

例如，QQ 有很多模块，登陆成功时用户名和密码存储在一个静态类中，这样访问新的模块时，模块自动访问静态类获取用户名和密码，避免每次进入时都需要重新输入用户名和密码。

释放资源 ( .NET 下 为 GC, Garbage Collection 垃圾回收器)

只有在程序全部结束后，静态类才会释放资源。

构造方法

这样初始化非常繁琐。构造方法帮助我们初始化方法。

注意

  1. 构造方法没有返回值，甚至不能写 void
  2. 构造方法的名称必须和类名一样字段

总结

  • 面向对象概念
  • 类
    • 语法
    • 成员：若不加访问修饰符，默认为 private
      • 字段：存储数据
      • 属性：保护字段，限定字段的取值和赋值，本质是 set 和 get 两个方法
      • 方法：描述对象的行为
      • 构造函数：初始化对象；没有返回值(没有 void), 名称与类名一样，实例化对象时调用
  • 对象
  • 关键字
    • new: 在内存中申请空间；创建对象；调用构造函数
    • this: 当前类的对象，和局部变量区分；显示调用构造函数
  • 静态和非静态
    • 调用上的区别：静态类通过类名调用，非静态类通过实例名调用

让 if 和 else 完美配合

若要执行 true 和 false 分支中的不同代码，请创建在条件为 false 时执行的 else 分支。

只有当条件的测试结果为 false 时，才执行 else 关键字后面的语句。

注意：if 和 else 语句下的缩进是为了方便读者阅读。 C# 语言忽略缩进或空格。 if 或 else 关键字后面的语句根据条件决定是否执行。本教程中的所有示例都遵循了常见做法，根据语句的控制流缩进代码行。

由于缩进会被忽略，因此需要使用 { 和 } 指明合适要在根据条件决定是否执行代码块中添加多个语句。通常建议对所有 if 和 else 子句使用这些大括号。

使用循环重复执行运算

while 语句检查条件，并执行 while 后面的语句。除非条件为 false，否则它会重复检查条件并重复执行这些语句。此示例引入了另一个运算符，counter 后面的 ++ 是增量运算符，它将 counter 值加 1, 并将计算后的值存储在 counter 变量中。

注意 请确保 while 循环条件在代码执行时确实能够切换为 false。 否则，创建的就是无限循环，即程序永不结束。

while 循环先测试条件，然后执行 while 后面的代码。do...while 先执行代码，然后再执行检查。

for 循环

此循环的工作原理与已用过的 while 循环和 do 循环相同。for 语句包含三个控制具体工作方式的部分，在示例中：

  1. for 初始值设定项：声明循环变量 counter, 并将其初始化为 0;
  2. for 条件：声明只要计数器的值小于 10, 此 for 循环就会继续执行；
  3. for 迭代器：指定在执行 for 语句后面的代码块后如何修改循环变量。示例中指定 counter 每次执行时递增 1.
