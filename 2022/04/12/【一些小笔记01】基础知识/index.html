转义符

\n 换行 (Windows 系统下是 \r\n)； \" 英文半角双引号；\t Tab 键；\b Backspace 键，在字符串两端没有效果；\\ 英文半角右斜线符号

@ 取消 \ 在字符串中的转译作用；将字符串按照原格式输出

类型转换

隐式类型转换

数据类型兼容且只能“小转大”，如 int -> double

显式类型转换

相互兼容的类型，如 int <-> double；

Convert 类型转换

对于不兼容的类型，需要使用 Convert 关键字，如 int -> string

注意 Console.ReadLine() 方法的返回值为 string 类型，一些情况需要类型转换。

异常捕获

异常：语法上没有错误，在程序运行中由于某些原因出现了错误导致程序无法正常运行。使用 try-catch 关键字可以捕获异常。

枚举

public, 访问修饰符，权限最低，作用域内任何地方都可以访问。
enum, 用于声明枚举的关键字
命名依照 Pascal 规范

为了规范，应当在命名空间内声明枚举。

结构

一次性声明多个不同类型的变量。

与枚举类似，同样写在命名空间下。

程序运行时，字段可以存储多个值，而变量只能存储一个值。
