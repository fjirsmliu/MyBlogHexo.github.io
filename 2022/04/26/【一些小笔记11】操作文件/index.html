文件操作

发表于2022-04-26更新于2022-05-03

Path 类

Path 在 System.IO 命名空间，是静态类，用于操作路径。

File 类

File 也在 IO 命名空间，也是静态类。

  • File 类一次性读取了文件的全部内容，所以只适合读小文件。对于大文件读取，需要使用文件流。

编码

编码：将字符串以什么形式存储为二进制？

ASC 128 位
ASCII 256 位

GB2312 中国大陆最早的编码，只包含简体字
Big5 包含繁体字的编码

Unicode 更统一且更全的编码，但解析慢，由 UTF32, UTF7, UTF8 组成
UTF-8 用于 Web 的编码，效率更高

乱码：通常是因为存储和读取文件采用的编码方式不一致。

绝对路径和相对路径

相对路径：文件相对于应用程序的路径。

开发中应尽可能使用相对路径。
