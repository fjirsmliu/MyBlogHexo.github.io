注意方法功能要单一，尽量不使用输入、输出函数保证代码更通用。

out, ref, params

out 参数

如果在一个方法中返回多个类型相同的值，可以考虑返回一个数组；然而需要返回值的类型不同时，可以考虑使用 out 参数，使用 out 参数在一个方法中可以返回多个不同类型的值。

ref 参数

要想输出增加奖金后的数值，通常需要设定函数的返回值类型为 int 使用 return 关键字，并定义一个参数接受返回值。能否有一种方法直接修改实际参数呢？

params 可变参数

如果不想声明 int[] s 这个变量，能否直接使用 Test("张三", 99, 88, 77); 呢？

这样就可以。

params 称为可变参数。params int[] 称为可变参数数组，但需要注意的是 C# 中的数组长度不可变，这是在调用函数的时候自动生成了一个 score[] = {99, 88, 77}.

具体来说，params 将实际参数列表中与可变参数数组类型一致的参数都处理为可变参数数组中的元素。

方法重载

这四个方法本质上都是在相加。能否写一个方法 M, 它可以填入不同的参数实现相加？
