在探索 C#（或任何编程语言）的过程中，可能会在编写代码时犯错。 编译器会发现并报告这些错误。 如果输出中有错误消息，请仔细比对示例代码和交互式窗口中的代码，看看要纠正哪些代码。 这样做有助于了解 C# 代码结构。

第一个示例展示了 if 和布尔类型的用途。布尔变量包含 true 和 false. C# 为布尔变量定义了特殊类型 bool. if 语句需要检查 bool 值，如果值为 true 则执行后面的语句，否则跳过。

让 if 和 else 完美配合

若要执行 true 和 false 分支中的不同代码，请创建在条件为 false 时执行的 else 分支。

只有当条件的测试结果为 false 时，才执行 else 关键字后面的语句。

注意：if 和 else 语句下的缩进是为了方便读者阅读。 C# 语言忽略缩进或空格。 if 或 else 关键字后面的语句根据条件决定是否执行。本教程中的所有示例都遵循了常见做法，根据语句的控制流缩进代码行。

由于缩进会被忽略，因此需要使用 { 和 } 指明合适要在根据条件决定是否执行代码块中添加多个语句。通常建议对所有 if 和 else 子句使用这些大括号。

使用循环重复执行运算

while 语句检查条件，并执行 while 后面的语句。除非条件为 false，否则它会重复检查条件并重复执行这些语句。此示例引入了另一个运算符，counter 后面的 ++ 是增量运算符，它将 counter 值加 1, 并将计算后的值存储在 counter 变量中。

注意 请确保 while 循环条件在代码执行时确实能够切换为 false。 否则，创建的就是无限循环，即程序永不结束。

while 循环先测试条件，然后执行 while 后面的代码。do...while 先执行代码，然后再执行检查。

for 循环

此循环的工作原理与已用过的 while 循环和 do 循环相同。for 语句包含三个控制具体工作方式的部分，在示例中：

  1. for 初始值设定项：声明循环变量 counter, 并将其初始化为 0;
  2. for 条件：声明只要计数器的值小于 10, 此 for 循环就会继续执行；
  3. for 迭代器：指定在执行 for 语句后面的代码块后如何修改循环变量。示例中指定 counter 每次执行时递增 1.

本教程未介绍另一个循环语句：foreach. froeach 语句为项序列中的每一项重复其语句。它最常用于集合，因此在下一教程中介绍。
